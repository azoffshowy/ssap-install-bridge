Local File Mode Is Blocked by the Browser

This app uses a JavaScript module (app.js). Modern browsers do not load modules from file:// pages. Start a small local web server and open the app via http://localhost:8080 or another HTTP(S) host instead. 

cd ssap-install-bridge
python3 -m http.server 8080

Then open in the browser: http://localhost:8080/docs/

webOS SSAP Install Bridge

Browser-based installer companion for LG webOS TVs. This view focuses on connection health, a small visual check, and a guided IPK delivery flow. On HTTPS pages, the app automatically prefers wss://<TV-IP>:3001.

Connection

Disconnected

On HTTPS pages, ws:// is mixed content and will be blocked. Because of that, the app automatically uses wss://<TV-IP>:3001. The TV certificate must be accepted manually in the browser once.
